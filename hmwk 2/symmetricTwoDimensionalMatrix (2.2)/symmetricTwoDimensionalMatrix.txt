Функция: Определить симметрична ли матрица относительно главной диагонали
    Сценарий: Вывод симметричности матрицы
        Дано просят ввести размер, а затем заполнить матрицу
                Когда пользователь вводит <size>, а затем <numbers>
                То должно вывести <result>
                Примеры:
                    | size | numbers | result                 |
                    | 2    | 1 5 5 1 | Матрица симметрична    |
                    | 2    | 5 0 5 0 | Матрица не симметрична |
                    
    Сценарий: Неправильный ввод размера
            Дано просят ввести размер матрицы
            Когда пользователь вводит <size>
            То должно вывести <error>
            Примеры:
                | size        | error                                                                                          |
                | 21000929293 | System.OverflowException: "Значение было недопустимо малым или недопустимо большим для Int32." |
                | f           | System.FormatException: "Входная строка имела неверный формат."                                |
                | -55         | System.OverflowException: "Переполнение в результате выполнения арифметической операции."      |

    Сценарий: Неправильный ввод элементов
            Дано просят ввести размер, а затем заполнить матрицу
            Когда вводят <size>, а затем <numbers>
            То должно вывести <error>
            Примеры:
                | size   | numbers       | error                                                                                          |
                | 3      | 3234567890    | System.OverflowException: "Значение было недопустимо малым или недопустимо большим для Int32." |
                | 5      | 1             | System.IndexOutOfRangeException: "Индекс находился вне границ массива."                        |
                | 2      | 1 2 3 4 5     | System.FormatException: "Входная строка имела неверный формат."                                |
                | 5      | f             | System.FormatException: "Входная строка имела неверный формат."                                |
                | 6      |               | System.FormatException: "Входная строка имела неверный формат."                                |
                | 2      | 2.2 2.2       | System.FormatException: "Входная строка имела неверный формат."                                |
