Функция: Получение количества различных чисел в одномерном массиве
    Сценарий: Вывод количества различных чисел
        Дано просят ввести размер массива, далее заполнить его
        Когда пользователь вводит <size>, а затем <numbers>
        То должно вывести <result>
        Примеры:
            | size | numbers     | result |
            | 5    | 0 1 2 1 2   |   3    |
            | 6    | 1 1 1 1 1 2 |   2    |
    
    Сценарий: Ошибка ввода размера одномерного массива
        Дано просят ввести размер массива
        Когда введен <size>
        То должно вывести <error>
        Примеры:
            | size | error                                                                                     |
            | -100 | System.OverflowException: "Переполнение в результате выполнения арифметической операции." |
            |  q   | System.FormatException: "Входная строка имела неверный формат."                           |
	    | 2.2  | System.FormatException: "Входная строка имела неверный формат."                           |
    
    Сценарий: Ошибка ввода элементов одномерного массива
        Дано просят ввести размер массива, а затем заполнить
        Когда введен <size>, а затем <numbers>
        То должно вывести <error>
        Примеры:
            | size | numbers | error                                                            |
            | 1    | 2.2     | System.FormatException: "Входная строка имела неверный формат."  |
            | 2    | 4 " "   | System.FormatException: "Входная строка имела неверный формат."  |
            | 5    | k       | System.FormatException: "Входная строка имела неверный формат."  |
            | 2    | " "     | System.FormatException: "Входная строка имела неверный формат."  |
